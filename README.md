# breast-MR_cytometry
To evaluate the diagnostic efficacy of radiomics features extracted from MR cytometry parameter mappings in differentiating benign and malignant breast tumors.
### extract_radiomics.py
Extract radiomics features from nii flies
### ML_mean_value.py
Bulid machine learning models based on mean values of feature mappings.
### ML_radiomics.py
Bulid radiomics-based machine learning models.
